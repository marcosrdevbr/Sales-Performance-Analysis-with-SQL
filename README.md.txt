# Sales Performance Analysis with SQL

## Project Overview

This project was developed to practice SQL for data analysis and business intelligence.

The goal was to analyze sales data and answer key business questions related to revenue, product performance, regional sales, profitability, and monthly performance.

## Dataset

The dataset contains sales information with the following columns:

- Date
- Region
- Category
- Product
- Sales
- Cost
- Quantity

## Business Questions

1. What is the total sales revenue?
2. Which region generated the highest revenue?
3. Which products sold the most units?
4. Which category generated the highest profit?
5. What is the average order value?
6. Which month had the best sales performance?

## Key Insights

- Total revenue was 6,900.
- North was the region with the highest revenue.
- Keyboard was the best-selling product.
- Electronics generated the highest profit.
- Average order value was 575.
- January 2025 had the best sales performance.
